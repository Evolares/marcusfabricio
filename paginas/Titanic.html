1. Definição do Problema:¶

O naufrgio Titanic um dos mais famosos da histria. Em 15 de abril de 1912, durante sua viagem inaugural, o Titanic afundou aps colidir com um iceberg, matando 1502 de 2224 passageiros e tripulantes. Essa tragdia sensacional chocou a comunidade internacional e levou a melhores regulamentaes de segurana para os navios.

Uma das razes pelas quais o naufrgio levou a tal perda de vidas foi que no havia botes salva-vidas suficientes para os passageiros e a tripulao. Embora houvesse algum elemento de sorte envolvido na sobrevivncia do naufrgio, alguns grupos de pessoas eram mais propensos a sobreviver do que outros, como mulheres, crianas e a classe alta.

Nesse desafio, construo um modelo preditivo que responde pergunta: "que tipo de pessoas tm mais probabilidade de sobreviver?" usando dados de passageiros (nome, idade, sexo, classe socioeconmica etc.)

2. Aquisição de dados:¶

Os dados foram divididos em dois grupos: conjunto de treinamento (train.csv) conjunto de teste (test.csv) O conjunto de treinamento foi usado para criar os modelos de aprendizado de mquina. Para o conjunto de treinamento, o resultado (tambm conhecido como "ground truth") para cada passageiro. Seu modelo baseia-se em "features", como sexo e classe dos passageiros. Usei a engenharia de recursos(feature engineering) para criar novas features. O conjunto de teste foi usado para avaliar o desempenho do modelo em dados no vistos. Para o conjunto de testes, no foi fornecido a varivel target para cada passageiro. Este o objetivo, avaliar se o passageiro sobreviveu ou no. Para cada passageiro no conjunto de testes, uso o modelo treinado para prever se eles sobreviveram ou no ao naufrgio do Titanic.

# FEATURE ENGINEERING: criao de features
for dataset in data_cleaner:    
    # nova feature: FamilySize
    dataset['FamilySize'] = dataset ['SibSp'] + dataset['Parch'] + 1
    
    #nova feature: IsAlone
    dataset['IsAlone'] = 1 #initialize to yes/1 is alone
    dataset['IsAlone'].loc[dataset['FamilySize'] > 1] = 0 # now update to no/0 if family size is greater than 1

    # nova feature: "Title"
    # Diviso rpida para "Name" com mtodo .split()
    dataset['Title'] = dataset['Name'].str.split(", ", expand=True)[1].str.split(".", expand=True)[0]


    # nova feature: FareBin, "cortamos os dados em 4(quartil)"; Segmentamos/classificamos em 4 grupos
    dataset['FareBin'] = pd.qcut(dataset['Fare'], 4)

    #nova feature: AgeBin para melhor distribuir os dados de idade por grupos, neste caso, 5. Segmentamos/classificamos em 5 grupos
    dataset['AgeBin'] = pd.cut(dataset['Age'].astype(int), 5)

def correlation_heatmap(df):
    _ , ax = plt.subplots(figsize =(14, 12))
    colormap = sns.diverging_palette(220, 10, as_cmap = True)
    
    _ = sns.heatmap(
        df.corr(), 
        cmap = colormap,
        square=True, 
        cbar_kws={'shrink':.9 }, 
        ax=ax,
        annot=True, 
        linewidths=0.1,vmax=1.0, linecolor='white',
        annot_kws={'fontsize':12 }
    )

    MLA_compare.loc[row_index, 'MLA Parametros'] = str(alg.get_params())
    
    # resultados com cross validation
    cv_results = model_selection.cross_validate(alg, data1[data1_x_bin], data1[Target], cv=cv_split, return_train_score=True)

    MLA_compare.loc[row_index, 'TEMPO UTILIZADO(mdia)'] = cv_results['fit_time'].mean()
    MLA_compare.loc[row_index, 'Acurcia mdia TREINO'] = cv_results['train_score'].mean()
    MLA_compare.loc[row_index, 'Acurcia mdia TESTE'] = cv_results['test_score'].mean()     

    # salvamos as predies, algoritmo por algoritmo
    alg.fit(data1[data1_x_bin], data1[Target])
    MLA_predict[MLA_name] = alg.predict(data1[data1_x_bin])
    
    row_index+=1
