REGRESSÃO POLINOMIAL

E se os dados forem realmente mais complexos do que uma simples linha reta?

Como utilizar um modelo linear para acomodar dados não lineares?

Basta acrescentar potencias de cada caracteristica como novas caracteristicas e, em seguida, treinar o modelo linear neste conjunto estendido de caracteristicas. Essa técnica é chamada de regressão polinomial.

from sklearn.metrics import mean_squared_error
from sklearn.model_selection import train_test_split

def plot_curvas_aprendizado(modelo, X, y):
    X_train, X_val, y_train, y_val = train_test_split(X, y, test_size=0.2)
    
    train_erros, val_erros = [],[]
    
    for m in range(1,len(X_train)):
        modelo.fit(X_train[:m], y_train[:m])
        y_train_predict = modelo.predict(X_train[:m])
        y_val_predict = modelo.predict(X_val)
        
        train_erros.append(mean_squared_error(y_train_predict, y_train[:m]))
        
        val_erros.append(mean_squared_error(y_val_predict,X_val))
        
    plt.plot(np.sqrt(train_erros), "r-+", linewidth=2, label="Conjunto de Treino")
    plt.axis([0, 80, 0, 7]) # [xmin, xmax, ymin, ymax]
    plt.plot(np.sqrt(val_erros), "b-", linewidth=3, label="Val")

from sklearn.linear_model import LinearRegression
Linear = LinearRegression()

plot_curvas_aprendizado(Linear, X, y)

Na medida em que adicionamos mais instancias, torna-se impossível que o modelo se ajuste perfeitamente, seja por conta de ruídos ou dados não lineares. Assim, atingirá um platô, um momento em que adicionar novas instancias ao conjunto de treinamento não tornará o erro médio muito melhor ou pior.

# e como fica um modelo polinomial de 10 graus.

from sklearn.pipeline import Pipeline

Polynomial_regression = Pipeline([
                                ("poly_features", PolynomialFeatures(degree=10, include_bias=False)),
                                ("Linear", LinearRegression()),
                                ])

plot_curvas_aprendizado(Polynomial_regression, X, y)

a) o erro nos dados de treinamento (Linha vermelha) é muito inferior ao encontrado no modelo de regressão linear b) Ao utilizar um conjunto de treinamento muito maior, as duas curvas continuariam se aproximando.
