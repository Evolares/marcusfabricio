CLASSIFICAÇÃO DE MANUSCRITOS

O conjunto de dados possui 70.000 de pequenas imagens de números escritos a mão por estudantes do ensino médio (Americano). O objetivo é treinar alguns modelos de Machine Learning que consiga aprender (e predizer) através das imagens e classificá-las corretamente.

ETAPAS DO PROJETO

Breve resumo de cada etapa do trabalho

Entendimento do Problema

O conjunto de dados possui 70.000 imagens de tamanho 28x28 (784 pixels) e a meta é treinar o modelo pra que possa identificar corretamente o manucristo.

Obtenção dos dados

As imagens podem ser baixadas em vários endereços da internet. Neste projeto, eles foram baixados pela função do sKlearn Fetch_OpenMl("mnist_784")

Criando dados para Teste

10.000 imagens foram reservadas (proporcionalmente) para testes. Os dados foram transformados para escla padrão (Standard Scaler)

Treinando o Modelo

Vários modelos foram treinados (SGDClassifier, RandomForestClassifier, Analise da Curva ROC). Após a avaliação do Precisão (Precision), Revocação (Recall) e Pontuação F1 (F1 Score) o SGDClassifier foi o que apresentou maior acurácia, sendo 90%.
